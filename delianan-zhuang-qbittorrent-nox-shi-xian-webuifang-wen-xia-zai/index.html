Delian安装 qbittorrent-nox 实现WEBUI访问下载

在 Debian 系统中安装 qbittorrent-nox 的步骤如下：

  1. 首先，打开终端。

  2. 更新你的包列表。在终端中输入以下命令：

sudo apt-get update
  1. 安装 qbittorrent-nox。在终端中输入以下命令：
sudo apt-get install qbittorrent-nox

这个命令会安装 qbittorrent-nox 和它的所有依赖。

  1. 安装完成后，你可以运行 qbittorrent-nox 来测试它是否安装成功。在终端中输入以下命令：
qbittorrent-nox

请注意，这个命令会在前台启动 qbittorrent-nox。你可以按 Ctrl+C 来停止它。

如果你希望在后台运行 qbittorrent-nox，你可以使用 -d 选项。例如：

qbittorrent-nox -d

这样，qbittorrent-nox 就会在后台运行，你可以关闭终端而不会停止它。

 

 

在 Debian 系统上，你可以使用 systemd 创建一个服务来在开机时自动运行 qbittorrent-nox。以下是步骤：

  1. 创建一个新的 systemd 服务文件。在终端中输入以下命令：
sudo nano /etc/systemd/system/qbittorrent.service
  1. 在打开的 nano 编辑器中，粘贴以下内容：
[Unit]
Description=qBittorrent Command Line Client
After=network.target

[Service]
# Do not change to "simple"
Type=forking
User=你的用户名
ExecStart=/usr/bin/qbittorrent-nox -d
ExecStop=/usr/bin/killall -w qbittorrent-nox

[Install]
WantedBy=multi-user.target

其中，将 "你的用户名" 替换为你希望运行 qbittorrent-nox 的用户的用户名。

  1. 保存并关闭 nano 编辑器。在 nano 编辑器中，你可以按 Ctrl+X，然后按 Y，最后按 Enter 来保存并退出。

  2. 设置 systemd 以在启动时启动 qbittorrent 服务。在终端中输入以下命令：

sudo systemctl enable qbittorrent.service
  1. 现在，你可以启动 qbittorrent 服务来测试它是否工作正常。在终端中输入以下命令：
sudo systemctl start qbittorrent.service
  1. 你可以使用以下命令检查服务的状态：
sudo systemctl status qbittorrent.service

如果一切正常，你应该会看到服务的状态是 "active (running)"。

这样，每次你的 Debian 系统启动时，qbittorrent-nox 就会自动运行。

 

在 qbittorrent-nox 中修改端口号，你需要通过其 Web 用户界面进行设置。以下是步骤：

  1. 启动 qbittorrent-nox。在终端中输入以下命令：
qbittorrent-nox

如果你已经设置了 systemd 服务，你可以使用以下命令启动它：

sudo systemctl start qbittorrent.service

  1. 打开你的 web 浏览器，访问 http://你的服务器IP地址:8080。这将打开 qbittorrent-nox 的 Web 用户界面。

  2. 在 Web 用户界面中，点击右上角的齿轮图标打开 "Options" 对话框。

  3. 在 "Options" 对话框中，点击 "Connection" 标签。

  4. 在 "Port used for incoming connections" 字段中，输入你希望使用的新端口号。

  5. 点击 "OK" 按钮保存你的设置。

请注意，如果你更改了端口号，你可能需要在你的防火墙中打开新的端口。在 Debian 系统中，你可以使用 ufw 命令来管理你的防火墙规则。例如，如果你希望打开端口 6881，你可以在终端中输入以下命令：

sudo ufw allow 6881

此外，如果你在家庭网络中运行 qbittorrent-nox，并且你的家庭网络有一个路由器，你可能需要在路由器的设置中进行端口转发。具体的步骤取决于你的路由器的型号和品牌。
